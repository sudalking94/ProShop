# 기능

- 회원가입
- JWT token 인증
- signal을 이용한 회원가입 전 name 변경
- 장바구니
- 상품검색
- 상품등록
- paypal 결제
- 상품관리 페이지
- 리뷰

# 사용 기술

- Django
- DRF
- React
- Redux

# 필수 설치 항목

- pipenv
- npm

# 설치 순서

```
$ git clone https://github.com/sudalking94/ProShop/
# 설치한 폴더로 이동
$ pipenv sync
$ pipenv shell
$ cd backend
$ python manage.py runserver
$ cd frontend
$ npm i
$ npm start
```
